Guide d'installation — extension Chrome + serveur OCR local

Prérequis 1. Installation 2. Extension Chrome 3. Utilisation Dépannage
✦

Prérequis

Outil Version minimale Vérification
git n'importe laquelle git --version
Python 3.10+ python3 --version
Google Chrome n'importe laquelle —
Node.js (optionnel) 18+ uniquement pour le formulaire de test local
ℹ️

Windows : utilisez le script PowerShell dédié (voir ci-dessous).

1

Lancer le script d'installation

macOS / Linux — télécharger et exécuter en une commande :

macOS 
curl -fsSL https://raw.githubusercontent.com/Nayarr/AssurFillOCR/main/setup_macos.sh | bash
Linux 
curl -fsSL https://raw.githubusercontent.com/Nayarr/AssurFillOCR/main/setup_linux.sh | bash

Windows — ouvrir PowerShell en tant qu'administrateur et exécuter :

PowerShell 
irm https://raw.githubusercontent.com/Nayarr/AssurFillOCR/main/setup_windows.ps1 | iex

Le script :

  • clone le dépôt dans ~/AssurFillOCR
  • crée un environnement virtuel Python isolé
  • installe flask, opencv-python, numpy, paddlepaddle, paddleocr
  • enregistre le serveur OCR comme service système (launchd sur macOS, systemd sur Linux) pour qu'il démarre automatiquement à chaque connexion
ℹ️

Le téléchargement des paquets peut prendre plusieurs minutes (~1 Go au total).

Au premier appel (première image analysée après installation), PaddleOCR télécharge ses modèles. C'est normal, ça ne se produit qu'une seule fois.

2

Charger l'extension dans Chrome

  1. Ouvrir Chrome → chrome://extensions
  2. Activer le mode développeur (interrupteur en haut à droite)
  3. Cliquer "Charger l'extension non empaquetée"
  4. Sélectionner le dossier ~/AssurFillOCR/extension

L'icône AssurFill apparaît dans la barre d'outils Chrome.

3

Utiliser l'extension

  1. Naviguer sur le formulaire de devis cible
  2. Cliquer sur l'icône AssurFill dans Chrome
  3. Déposer les photos du permis de conduire et/ou de la carte grise (recto/verso)
  4. Cliquer "Lancer l'analyse" — le serveur OCR traite les images (~5-15 secondes)
  5. Cliquer "Injecter" pour remplir le formulaire automatiquement
?

Résolution de problèmes

"Impossible de joindre le serveur OCR"

Le service n'a pas démarré. Vérifier son état :

bash 
# macOS
launchctl list | grep assurfill

# Linux
systemctl --user status assurfill-ocr

Pour le démarrer manuellement :

bash 
~/AssurFillOCR/.venv/bin/python ~/AssurFillOCR/extension/ocr_server.py

Le bouton "Injecter" ne remplit rien

S'assurer que l'onglet actif contient bien le formulaire cible et que la page est entièrement chargée.

Relancer le script après une mise à jour

setup.sh est idempotent : le relancer met à jour le dépôt et réinstalle le service proprement.

bash 
bash ~/AssurFillOCR/setup.sh
